Comandos iniciais do git 

git init = iniciar um repositorio 
git add . = adicionar os arquivos atuais ao proximo commit
git status = verificar o status atual do repositorio git 
git commit -m "mensagem do commit" = criar um novo commit com uma mensagem
git push = enviar as atualizaçoes para a nuvem na branch que esta ativa atualmente
git branch = permite listar e ver qual branch esta ativa atualmente
git checkout (nome-da-branch) = permite mudar para uma nova branch 
git reflog = verificar historico de atualizações

Comandos avançados(para trabalho em equipe)

git reset --hard = navegar entre as versoes do codigo 
git branch staging = criar uma branch para receber atualizações nao testadas
git pull = atualizar branch atualmente ativa
git merge "branch a receber merge" = permite fazer unir branch ativa atualmente com outra branch
git checkout -b nome-da-branch (branch principal) = permite mudar e criar uma nova branch com base em outra
touch .gitignore = ignorar arquivos para o controle de versionamento
